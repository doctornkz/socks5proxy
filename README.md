# Usage
```
git clone https://github.com/doctornkz/socks5proxy.git
```
#### Change constants
```
const (
	login    = "CHANGE_ME"
	password = "CHANGE_ME"
	ipport   = "0.0.0.0:1234"
	timeout  = 10
)
```

#### Save and build
```
go build
```

#### Run
```
./socks5proxy
```
